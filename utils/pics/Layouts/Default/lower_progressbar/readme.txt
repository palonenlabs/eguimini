This folder contains graphics for the lower graph in the eGUI mini. It's purpose is to display values between 0 and 100 % of full-scale.
 
Picture format should be:

size 96 x 16 px
2 bit color (black and white)

File names should be pb-0.png, pb-1.png, â€¦ ,pb-100.png representing values between 0 and 100 %. They are scaled to actual proper values in each type of data shown in the display. Each type will have  it's own full-scale max-reading configured in the configuration of the eGUI mini.
